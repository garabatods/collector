# Collector App — Step-by-Step Build Plan

## Phase 0 — Setup
1. Verify Flutter + VS Code environment
2. Create the Flutter project
3. Connect the app to Supabase

## Phase 1 — App foundation
4. Create folder structure
5. Add routing
6. Add theme and app shell
7. Add placeholder tabs/screens

## Phase 2 — Authentication
8. Implement Supabase auth
9. Handle session-aware entry flow
10. Add sign in / sign up / sign out

## Phase 3 — Database design
11. Create initial Postgres schema
12. Add RLS policies
13. Create Flutter models and repositories

## Phase 4 — Collection CRUD
14. Build collection list screen
15. Build add collectible flow
16. Build detail screen
17. Build edit flow
18. Add delete with confirmation

## Phase 5 — Photos
19. Prepare Supabase Storage strategy
20. Add image picker
21. Upload and display one primary photo per collectible

## Phase 6 — Collector features
22. Add wishlist
23. Add search
24. Add filters
25. Add simple dashboard stats

## Phase 7 — Polish
26. Improve loading states
27. Improve error states
28. Improve empty states
29. Create QA checklist
30. Prepare release-readiness checklist

## Working method
- Complete one step at a time
- Test after each step
- Do not move to the next step until the current one is working
